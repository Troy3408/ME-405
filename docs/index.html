By: Troy Feinstein and Michael Hartley

Project Description:

The goal of this project was to create a Line following Maze Solver based off of a Romi Robot Kit using a Nucleo-STM32. Our robot will be placed in the START section and wait for the user to activate it with the push button on the Nucleo. Once activated, the robot will follow the line through the entire course until it reaches the FINISH Section, in which it will stop, turn around, and make its way back to the the START Section. While traversing the course, the Romi will have to deal with tight turns, obstructing perpendicular lines, curved dashed lines, and it will have to detect and go around a random object that will be blocking the course.

For this project, we are using Python, with the Micropython, Pyb, and Machine libraries. All relevant source file can be found on the github repository under the Term Project Folder: https://github.com/Troy3408/ME-405
